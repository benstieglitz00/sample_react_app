# Simple Text Box Project

This project is a simple web application that includes a text box and a button. When the button is clicked, a message is displayed in the text box.

## Project Structure

```
simple-textbox-project
├── src
│   ├── index.html      # HTML structure of the application
│   ├── styles.css      # Styles for the application
│   └── app.js          # JavaScript code for handling button click
└── README.md           # Documentation for the project
```

## Getting Started

To run this application, follow these steps:

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser.

## Dependencies

This project does not have any external dependencies. It uses standard HTML, CSS, and JavaScript.

## Usage

- Enter text in the text box.
- Click the button to display a message in the text box. 

Feel free to modify the styles in `styles.css` or the functionality in `app.js` to suit your needs.
